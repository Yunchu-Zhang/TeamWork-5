# LangSAM 优点总结

## 1. 文本输入方便

LangSAM 最大的优点是用户只需要输入英文目标词，不需要手动点选目标位置。例如输入 `person.`、`wheel.`、`cup.`，模型会先根据文本找到目标，再自动生成分割 mask。

这对普通用户更友好，因为用户不需要理解坐标、点选正负样本、mask 后处理等细节。

## 2. 适合目标名称明确的场景

当图片里的目标比较常见、名称明确时，LangSAM 的效果比较稳定。例如：

| Prompt | 适合场景 |
| --- | --- |
| `person.` | 人物、行人、半身或全身照 |
| `car.` | 汽车整体 |
| `wheel.` | 轮胎、车轮 |
| `cup.` | 杯子、水杯 |
| `book.` | 书、本子 |
| `chair.` | 椅子 |
| `bottle.` | 瓶子 |

## 3. 不需要用户点击目标

相比 SAM2 点选分割，LangSAM 不要求用户点击图像中的目标。前端只需要提供：

- 一张图片
- 一个英文 prompt

后端就可以调用 LangSAM 自动完成文本定位和分割。

## 4. 适合接入 `/segment/lang`

LangSAM 的输入输出结构比较清晰，适合封装成后端接口：

| 输入 | 输出 |
| --- | --- |
| 图片文件 | mask |
| 英文 prompt | 检测框 |
| 阈值参数 | 标签和分数 |

因此它适合作为文本分割接口 `/segment/lang` 的算法核心。

## 5. 可以一次识别多个目标

LangSAM 支持多个英文 prompt，例如：

```text
kiwi. watermelon. blueberry.
```

这种方式适合一张图里有多个不同类别目标的情况，可以减少用户多次操作。
