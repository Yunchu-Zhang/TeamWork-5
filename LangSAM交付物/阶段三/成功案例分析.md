# 成功案例分析

## 1. 汽车轮胎案例

对应截图：

```text
LangSAM交付物\阶段一\案例截图\成功案例1_car_wheel.png
LangSAM交付物\阶段二\文本分割样例\样例1_car_wheel_output.png
```

输入 prompt：

```text
wheel.
```

效果好的原因：

- 轮胎是常见物体，英文 prompt 与目标类别匹配度高。
- 轮胎和车身、地面之间边界明显，SAM2 容易分割出完整形状。
- 图片中目标尺寸较大，不是远处小目标。
- 背景虽然有树木，但轮胎位于汽车主体上，语义关系比较明确。

## 2. 水果多目标案例

对应截图：

```text
LangSAM交付物\阶段一\案例截图\成功案例2_fruits.png
LangSAM交付物\阶段二\文本分割样例\样例2_fruits_output.png
```

输入 prompt：

```text
kiwi. watermelon. blueberry.
```

效果好的原因：

- prompt 都是具体英文名词，不是抽象描述。
- 水果颜色差异明显，模型容易根据视觉特征区分。
- 主体集中在画面中间，目标没有被大量背景遮挡。
- 多个 prompt 用英文句点分隔，符合 LangSAM 的文本输入习惯。

需要注意：

- 多目标情况下，标签和框可能有重叠。
- 如果后端要生成透明 PNG，需要决定是合并所有 mask，还是分别保存每个目标。

## 3. 人物案例

对应截图：

```text
LangSAM交付物\阶段一\案例截图\成功案例3_person.png
LangSAM交付物\阶段二\文本分割样例\样例3_person_output.png
```

输入 prompt：

```text
person.
```

效果好的原因：

- `person.` 是模型非常常见的类别，识别稳定性较高。
- 人物在画面中占比较大，轮廓比较完整。
- 人物和背景颜色、形状差异明显。
- prompt 简短明确，不会造成语义歧义。

## 4. 总结

成功案例通常满足这些条件：

- 目标名称明确，能用简单英文名词表达。
- 目标在图中比较大，边界清楚。
- 背景干扰少，目标没有严重遮挡。
- 同类目标数量不太多，模型不容易选错。
