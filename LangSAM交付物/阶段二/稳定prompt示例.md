# 稳定 prompt 示例（5-10 个）

建议使用英文名词
| Prompt | 稳定性 | 备注 |
| --- | --- | --- |
| `person.` | 高 | 常见类别，通常较稳定 |
| `car.` | 高 | 车辆整体 |
| `wheel.` | 高 | 轮胎/车轮 |
| `dog.` | 中高 | 主体清晰时稳定 |
| `cat.` | 中高 | 主体清晰时稳定 |
| `cup.` | 中高 | 透明可能不稳 |
| `chair.` | 中高 | 多椅子场景可能选错 |
| `bottle.` | 中高 | 透明瓶或遮挡可能不稳 |
| `phone.` | 中 | 小目标可能不稳 |
| `book.` | 中 | 与背景相近时可能不稳 |
