
# 一、阶段一：基础准备与算法验证

## 阶段一目标

这一阶段不是做完整系统，而是先证明每一块能单独跑起来。

最终要做到：

> 有测试图片、有 LangSAM 运行结果、有 SAM2 点选初步结果、有后端上传接口、有前端页面雏形、有报告目录。

### 详细产出

### 详细产出

```json
{
  "status": "success",
  "method": "point",
  "original_url": "/uploads/test.jpg",
  "mask_url": "/masks/test_mask.png",
  "png_url": "/pngs/test_object.png"
```
